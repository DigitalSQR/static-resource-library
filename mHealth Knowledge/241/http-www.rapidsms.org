    <div><h1>ABOUT RAPIDSMS</h1>
    <p class="lead">
      RapidSMS is a free and open-source framework for rapidly building mobile services for scale. RapidSMS is built with <a href="http://python.org/">Python</a> and <a href="http://djangoproject.com">Django</a> and is designed for building robust, highly customized mobile services with web-based dashboards. RapidSMS provides a flexible platform and modular components for large-scale data collection, managing complex workflows, and automating data analysis.
    </p></div>
